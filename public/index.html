Random Quotes API

JSON HTTP API for random and filtered quotes. All routes are read-only (GET / OPTIONS).

Endpoints

  • /api Discovery — lists routes
  • /api/health Load status and dataset info
  • /api/quotes/random Random quote(s); try ?limit=3
  • /api/quotes Filtered random set — see repo README for params
  • /api/categories Categories; add ?counts=true
  • /api/authors Author list; filter with ?q=name
Open the JSON endpoints above directly or from your app. Full query parameters are documented in the repository README.
